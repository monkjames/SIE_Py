# Getting Started
- create a folder for storing all the string (.stf) files from SIE
it can be easiest to make this folder inside this notebook but you'll want to add it to git ignore if you plan to use GIT

- create a folder for storing all the excel files
it can be easiest to make this folder inside this notebook but you'll want to add it to git ignore if you plan to use GIT

- use SIE to export all STF files to the folder you created

- edit config.py, follow instructions in that file. You will need the paths from the above steps
